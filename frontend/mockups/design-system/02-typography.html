Decision 02 of 04

Typography

Three font pairings — each one changes the entire personality of the app. The same colors and components are used across all three so you can isolate the type decision.

All three use JetBrains Mono for code/card-identifier text (not up for debate — it's the clearest option)
Option A
Space Grotesk — headings Inter — body

Clean, modern, precise. The current direction. Works extremely well for data-heavy interfaces. No atmosphere — all clarity.
