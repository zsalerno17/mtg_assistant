Chart 1 of 3
Mana Curve
Non-land card count by mana value. Both variations keep the vertical bar histogram — the differences are in color strategy and annotation style. The avg CMC reference line is always included.
Variation A — Zone Colors Recommended

CMC 0–2 = acceleration (primary blue) · 3–4 = spell density (secondary gold) · 5+ = haymakers (danger red). Three zones give instant visual read of deck tempo. Count labels above each bar replace Y-axis noise.

Variation B — Single Accent Alternative

All bars in primary blue at uniform opacity. Cleaner/more minimal — lets the shape speak for itself. Count labels still shown. Avg CMC dashed reference line.

Chart 2 of 3 — replaces card type donut
Role Composition
What each card actually does in the deck. Based on computed role counts already returned by the backend. "Threats/Other" is a derived remainder bucket. Semantic MTG colors per role.
Variation A — Horizontal Bars Recommended

One bar per functional role. Bar length = proportion of deck. Count at right edge. Labels along Y-axis (left). Easiest to scan; natural for ranking/comparison. BI standard.

Variation B — Stacked Bar + Legend Alternative

All roles in a single proportional bar. Legend below with counts. Very compact — good for small screen. Harder to compare segment sizes precisely. More visual/decorative.
