Implementation Checklist — Desktop (≥768px)

  • Screen size: This spec applies to ≥768px only. Below 768px, switch to Direction B (Hero Editorial) mobile spec.
  • Commander hero: Horizontal layout, 120×168px art, Cinzel 32px name, gradient background, blue border
  • Stats grid: Auto-fit minmax(140px, 1fr), 48px stat values (BIG), color-coded good/warning/bad
  • Weaknesses section: Full-width dedicated area, red tinted background, 3-column grid (responsive to 2→1)
  • Charts: Two-column layout (1fr 1fr), stacks to single column below 1200px
  • Typography: Cinzel ONLY for commander name. Inter everywhere else (tabs, labels, descriptions).
  • Tabs: Active state with blue underline, 150ms transition, Inter font
  • Max-width: 1400px container for desktop information density
  • Colors: Approved palette tokens (--accent-primary, --accent-danger, --surface, --border)
  • Accessibility: Focus rings, semantic HTML, ARIA labels where needed
